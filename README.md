# Shivam – AI & ML Portfolio

This is a minimal, modern personal portfolio for showcasing AI and ML projects such as multi‑tool agents, computer vision apps, and ML APIs.

## Structure

- `index.html` – main single‑page portfolio
- `styles.css` – layout and visual design
- `script.js` – theme toggle, smooth scrolling, and year

## Running locally

You can use any static server or even just open `index.html` in your browser.

With Node.js installed:

```bash
cd "c:\personal by shivam\portfolio 2"
npm install serve -g
serve .
```

Or using the script in `package.json` (will use `npx serve`):

```bash
cd "c:\personal by shivam\portfolio 2"
npm run start
```

Then open the printed local URL (for example `http://localhost:3000`) in your browser.

## Customization

- Update your name, summary, and project descriptions in `index.html`.
- Replace the placeholder email and social links in the contact section.
- Tweak colors, spacing, or layout in `styles.css`.
